Projetos

  • PixelsArt

    Uma página web que contém uma paleta de cores funcional para criar desenhos em pixels.

    Tecnologias utilizadas: JavaScript, CSS e HTML.

  • Facebook SignUp

    Recriando a antiga página de inicial/login do Facebook, procurando ser o mais parecido possível.

    Tecnologias utilizadas: JavaScript, CSS e HTML.

    Este projeto foi realizado em pair programming com Fernanda Porto.

  • Star Wars Planets Search

    Aplicação com uma lista com filtros de planetas do universo de Star Wars.

    Tecnologias utilizadas: React Context API e Hooks.

  • Trybe Wallet

    Uma carteira de controle de gastos com conversor de moedas, ao utilizar essa aplicação um usuário deverá ser capaz de: adicionar, remover e editar um gasto, visualizar uma tabelas com seus gastos e visualizar o total de gastos convertidos para uma moeda de escolha.

    Tecnologias utilizadas: React e Redux.

  • Online Store

    Uma versão simplificada, sem persistência no banco de dados, de uma loja online onde os usuários poderão: buscar produtos a partir da API, adicioná-los e removê-los do carrinho de compras, visualizar detalhes, avaliações e finalizar a compra (simulada).

    Tecnologias utilizadas: Métodos Ágeis, Kanban e React.

    Este projeto foi realizado em grupo com Alessandra Rezende, Arthur Menezes, Emerson Saturnino e Lucas Portella.

  • Trivia

    Um jogo de perguntas e respostas baseado no jogo Trivia o qual o usuário poderá: configurar os temas e quantidade de perguntar, fazer login, verificar sua pontuação e visualizar a página de ranking.

    Tecnologias utilizadas: Kanban, React e Redux.

    Este projeto foi realizado em grupo com Dennis Marcelo, Fernanda Porto e Guilherme Caixeta.

  • Bonna Mango

    Um app de receitas onde é possível ver, buscar, filtrar, favoritar e acompanhar o processo de preparação de receitas e drinks! A base de dados são 2 APIs distintas, uma para comidas e outra para bebidas. O layout tem como foco dispositivos móveis.

    Tecnologias utilizadas: Kanban, React Context API, Hooks e Bootstrap.

    Este projeto foi realizado em grupo com Aline Barbosa, Luiz Reina e Pedro Henrique.

  • SpotifyClone

    Neste projeto, simulando de uma forma silplificada, foi utilizado as habilidades de: modelar um banco de dados aplicando a 1ª, 2ª e 3ª Forma Normal, criar tabelas, lidar com VIEWs, alterar tabelas existentes e criar reações dinâmicas com TRIGGERS.

    Tecnologia utilizada: MySQL.
